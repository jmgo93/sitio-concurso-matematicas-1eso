# Sitio del concurso · Matemáticas 1.º ESO

Este repositorio contiene solo la versión estática lista para publicar del sitio del concurso.

Web publicada:

- https://jmgo93.github.io/sitio-concurso-matematicas-1eso/

Incluye:

- `instrucciones-jurado.pdf`: guía de apertura y navegación para el jurado.
- `index.html`: portada general del portal.
- `buenas-practicas.html`: página del reconocimiento como Buena Práctica Educativa.
- `tiendas/`: proyecto `Tiendas de Barrio`.
- `indalo/`: proyecto `La Tierra del Indalo`.
- `assets/`: estilos, scripts, imágenes y recursos comunes.

No incluye:

- archivos fuente de trabajo fuera del portal,
- scripts de generación,
- materiales auxiliares que no son necesarios para visualizar la web.

La estructura está pensada para publicarse directamente con GitHub Pages desde la raíz del repositorio.
